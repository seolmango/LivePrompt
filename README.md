# LivePrompt

실시간 동기화되는 프롬프트. 곡, 유저를 추가하고 셋 리스트를 설정하여 각 유저가 볼 화면(가사, 악보)를 설정할 수 있습니다.

## 사용법

```cmd
pip install -r requirements.txt
flask db init
flask db migrate
flask db upgrade
flask run
```
